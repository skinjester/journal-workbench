A map of how work moved over time

Since 2013, I've been keeping timestamped notes about the projects I work on as plaintext files kept open and added to over the course of a day. Over time I've used different ways of entering, syncing, and merging that material across multiple machines, and the practice has become second nature. I always knew there was intelligence captured in those files, but the detail was so granular that it felt impossible to really extract. This view turns 118 monthly summaries into a timeline: each row is a project, each column is a month, and brighter cells mean a project was more active in that stretch.

Projects + Milestones

This is more than a simple timeline. It is an activity map: a way to see which projects were primary, secondary, parallel, or dormant, and when prototypes, presentations, or PDFs appeared as outputs. Scroll the timeline, zoom with the slider, and open a project row to inspect the work underneath it without losing the larger picture.
